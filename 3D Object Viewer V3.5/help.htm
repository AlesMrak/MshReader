3D Object Viewer Help

3D Object Viewer allows you to view various 3D models. The viewer can load a range of models including DirectX, 3DS, BSP maps and heightmap terrains. Viewing controls are simple, right click and hold on a 3D view to rotate or hold both mouse buttons to zoom. There are two camera modes, orbit and explore. Use orbit to rotate quickly around the 3D object or explore for roaming around a bsp map or terrain. When using explore left click to move forward and hold shift to accelerate.

 

File Menu

Load Model: Load a 3D model. The formats below can be opened.

  • .x - DirectX model
  • .3ds - 3D Studio model
  • .dbo - DarkBASIC objectl (U5.8)
  • .md2 - Quake2 Model

    • Load DirectX Mesh: Load a DirectX mesh.

    Load BSP: BSP files are usually used for indoor maps in games.

    Load Terrain: To load a terrain, you will need to load three seperate image files. A heightmap image, a terrain texture and a detail texture.

    Save as DBO Object: Save the current model as a dbo object. This is the native format for DarkBASIC Pro.

    Save as DirectX Mesh: Save the current model as a DirectX mesh. This format will not contain any animation data.

    DBO Batch Convert: Batch convert multiple models to the DarkBASIC .dbo format.

    Save Screenshot: Save a .jpg screenshot.

     

    Settings Dialog

    View Range: This adjusts the clipping distance. Can be increased if you can't see the whole of the object.

    Floor Size: This is the size of the floor plain. It's used to see axis orientation and where the object is to the 0,0,0 origin point.

    Ambient Color: The level of ambient light. 0-100 Where 0 is dark and 100 is bright. 50 is a good medium.

    Change Background Colour: The background color can be changed if it clashes with the objects color.

     

    View Menu

    Toggle Floor: The floor plane can be solid, ghosted or hidden completely.

    Toggle Wireframe: View the objects wireframe.

    Toggle Rotation: Rotate the object automatically.

    Toggle Light: Switch light on or off.

     

    Resize Object Dialog

    You can resize the entire object by adjusting the scale or size values here.

     

    Animation Dialog

    You can see an object animating between it's start and end frames. Click Play to play the animation once or Loop to play it continuously.

     

    Limb Control Dialog

    Some models are made up of different limbs. You can select a limb and adjust it's scale, rotation and offset from it's original position here. The selected limb will flash on the model.
